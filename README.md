## 使用方法

下载项目到本地，安装依赖

```
npm install
```

将.env环境变量配置文件中的相关信息填入

```
REACT_APP_APPID = '你的appId'
REACT_APP_APISecret = '你的apiSecret'
REACT_APP_APIKey = '你的apiKey'
REACT_APP_Uid = '你的uid(这个可以随便填)'
```
运行项目
```
npm start
```
